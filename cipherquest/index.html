CipherQuest

Decode the Mystery

About

CipherQuest is a cryptography puzzle game that challenges you to crack ciphers and unlock secrets. Journey through 10 chapters and 100 puzzles as you learn classic encryption techniques while unraveling an intriguing mystery.

Cipher Types

  • Caesar Cipher — Each letter is shifted by a fixed number
  • Hidden Caesar — Find the shift pattern yourself
  • Variable Caesar — The shift changes within the message
  • Reverse Cipher — The message is written backwards
  • Atbash Cipher — A becomes Z, B becomes Y, and so on
  • Mixed Cipher — Multiple techniques combined
  • Multi-Step — Decode the message, then decode it again

How to Play

  • Read the encrypted message carefully
  • Use the hints if you get stuck
  • Enter your decoded answer when ready
  • Complete puzzles to progress through the chapters and uncover the story
